Reading guide

How to interpret this tab

  • NodeDB favorite means the source node already marks that direct neighbor as favorite in its own `nodesInMesh` snapshot.
  • Managed favorite is Meshonator's own flag on the managed target node.
  • Mutual visibility means node A sees node B at 0 hop and node B also sees node A at 0 hop.
